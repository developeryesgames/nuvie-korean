## 설명란

### 한글
```
울티마 6: 거짓 예언자 (Ultima VI: The False Prophet) 한글화 패치입니다.

1990년 오리진 시스템즈에서 발매된 고전 RPG 명작을
AI(Claude)를 활용하여 약 40시간 만에 전체 한글화했습니다.

■ 번역 내용
- NPC 대화: 200명 이상, 7,700줄 이상
- 아이템/오브젝트: 600개 이상
- 마법 주문: 121개 전체
- 책/표지판/두루마리: 127개
- UI, 전투 메시지, 시스템 메시지 전체
- 인트로/엔딩 시퀀스

■ 특징
- 한글 키워드 입력 지원 (예: "이름", "일", "합류" 등)
- 한글 조사 자동 처리 (을/를, 이/가, 은/는)
- 16x16, 32x32 한글 비트맵 폰트

■ 필요 사항
- 울티마 6 정품 (GOG 등에서 구매 가능)
- Nuvie 엔진 기반

■ 다운로드
GitHub: [링크]

■ AI 활용 워크플로우
1. 원본 대사 데이터 추출 (Python 스크립트)
2. AI에게 세계관/말투 컨텍스트 제공
3. 대화별 번역 및 검수
4. 엔진 코드 수정 (한글 폰트, 키워드 시스템)
5. 테스트 및 수정

이 프로젝트는 AI를 활용한 게임 로컬라이제이션의 가능성을 보여주는
개인 프로젝트입니다.
```
