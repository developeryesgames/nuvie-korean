## 설명란

울티마 6: 거짓 예언자 (Ultima VI: The False Prophet) AI 기반 한글화 패치 v1.0

1990년 오리진 시스템즈의 명작 RPG '울티마 6'를 최신 AI 기술(Claude Opus 4.5 등)을 활용하여 약 40시간 만에 전체 한글화 및 시스템 최적화를 완료했습니다. 본 프로젝트는 1인 개발자의 AI 활용 생산성 혁신과 고전 게임 복원 가능성을 확인하기 위한 기술 실증 프로젝트입니다.

■ 주요 번역 현황
NPC 대화: 200명 이상의 NPC, 약 7,700라인 이상의 방대한 텍스트 완역
데이터베이스: 아이템/오브젝트 600여 개, 마법 주문 121개 전체 번역
문서 콘텐츠: 인게임 서적, 표지판, 두루마리 등 127종 번역
인터페이스: UI, 전투/시스템 메시지, 인트로/엔딩 시퀀스 전체 한글화

■ 기술적 구현 특징
한글 키워드 입력 시스템: 원작의 키워드 시스템을 분석하여 한글 입력(예: "이름", "일", "합류") 및 별칭(Alias) 지원
동적 조사 처리 엔진: 한국어 특유의 조사(을/를, 이/가, 은/는)를 문맥에 맞게 자동 처리하는 로직 구현
커스텀 가독성 폰트: 저해상도 환경에 최적화된 16x16, 32x32 한글 비트맵 폰트 자체 제작 및 렌더링 수정

■ AI 활용 워크플로우 (Tech Stack)
Data Extraction: Python 스크립트를 통한 원본 바이너리/텍스트 데이터 추출
Context Injection: AI 모델에게 울티마 세계관, 캐릭터별 말투, 고어(Archaic English) 컨텍스트 주입
AI Translation: Claude Opus 4.5 기반의 일괄 번역 및 문맥 검수
Engine Modding: 오픈 소스 엔진(Nuvie) 소스 수정을 통한 한글 폰트 엔진 및 키워드 파서(Parser) 구현

■ 저작권 및 이용 안내 (Disclaimer)
본 프로젝트는 상업적 목적이 없는 비영리 팬 메이드 프로젝트입니다.
Ultima VI의 저작권은 원저작권자인 **Electronic Arts (EA)**에 있습니다.
본 패치는 게임 데이터 파일을 포함하지 않으며, **울티마 6 정품(GOG, EA 앱 등)**이 설치된 환경에서 Nuvie 엔진을 통해 구동됩니다.

Ultima VI: The False Prophet - AI-Assisted Complete Korean Localization
A solo developer's technical demonstration of fully translating the 1990 classic RPG into Korean in just 40 hours, leveraging advanced AI models (Claude 3.5 Sonnet, Gemini, etc.).
Key Features:
Full Translation: 200+ NPCs, 7,700+ dialogue lines, all items, spells, and documents.
Enhanced UX: Full Korean keyword input support and dynamic particle (Grammar) handling.
Tech Stack: Automated data extraction via Python, AI-driven context-aware translation, and Nuvie engine modification for font rendering.
Note: This is a non-commercial fan project. Requires an original copy of Ultima VI to function. All rights to Ultima VI belong to Electronic Arts (EA).
