■ 소개
--------------------------------------------------------------------------------
1990년 오리진 시스템즈에서 발매된 명작 RPG "울티마 6"의 한글화 패치입니다.
Nuvie 엔진을 기반으로 제작되었습니다.


■ v1.5.1 패치노트
--------------------------------------------------------------------------------
[버그 수정]
- NPC 스케줄러 오류 수정
  · 지켜보고 있을 때 NPC가 계속 자는 문제 해결
  · 원본 U6 동작과 동일하게 스케줄 항상 적용

- 야영 시 게임이 멈출 수 있는 문제 수정
  · 파티 이동 중 중복 이동 요청 방지

- 멀티타일 시체(사이클롭스 등) Z-order 렌더링 버그 수정
  · 2x2 크기 몬스터 시체의 확장 타일이 캐릭터 위에 그려지던 문제 해결

- 던전 진입 시 크래시 버그 수정
  · TimedPartyMove 메모리 관리 개선

[번역 수정]
- 일부 시스템 메시지 자연스럽게 수정


■ v1.5 패치노트
--------------------------------------------------------------------------------
[새 기능]

- 시스템 메시지 번역 보완
- 월드맵에서 우클릭 기능 설명 추가


■ 번역 내용
--------------------------------------------------------------------------------
- NPC 대화: 200명 이상, 7,700줄 이상
- 아이템/오브젝트 설명: 600개 이상
- 마법 주문 이름: 121개 전체
- 책, 표지판, 두루마리: 127개
- UI 텍스트, 전투 메시지, 시스템 메시지 전체
- 인트로/엔딩 시퀀스


■ 특징
--------------------------------------------------------------------------------
● 한글 키워드 입력 지원
  NPC와 대화할 때 한글로 키워드를 입력할 수 있습니다.
  예: "이름", "일", "합류", "안녕" 등 (770개 이상 지원)

● 한글 조사 자동 처리
  받침 유무에 따라 을/를, 이/가, 은/는이 자동으로 적용됩니다.

● 전용 한글 폰트
  16x16, 32x32 비트맵 한글 폰트가 포함되어 있습니다.


■ 필요 사항
--------------------------------------------------------------------------------
- 울티마 6 정품 게임 파일
  (GOG.com 등에서 구매 가능: https://www.gog.com/game/ultima_6)
- Windows 10/11
- (선택) MT-32 녹음 MP3 음악 파일 (고음질 배경음악용)


■ 설치 방법
--------------------------------------------------------------------------------
1. 이 패치 파일의 압축을 해제합니다.

2. ULTIMA6 폴더에 정품 게임 파일을 복사합니다.
   필요한 파일:
   - CONVERSE.A
   - CONVERSE.B
   - MAP
   - OBJLIST
   - 기타 게임 데이터 파일들

3. nuvie.exe를 실행합니다.

4. 한글 모드가 기본으로 활성화되어 있습니다.


■ 설정 변경
--------------------------------------------------------------------------------
nuvie.cfg 파일을 편집하여 설정을 변경할 수 있습니다.

한글 모드 끄기:
<korean_mode>false</korean_mode>


■ MP3 음악 설정 (선택사항)
--------------------------------------------------------------------------------
울티마 6 원본은 AdLib/FM 음악을 사용하지만, MT-32로 녹음된 고음질 MP3 음악을
사용할 수 있습니다.

[MP3 음악 다운로드]
Dor-Lomin MT-32 Archive에서 무료로 다운로드할 수 있습니다:
https://github.com/Dor-Lomin/MT-32-Music-Archive

필요한 파일 (Ultima VI 폴더):
- opening.mp3 (시작화면)
- introduction.mp3 (인트로)
- createcharacter.mp3 (캐릭터 생성)
- ultimatheme.mp3 (울티마 테마)
- rulebritannia.mp3 (룰 브리타니아)
- wanderer.mp3 (필드 음악)
- stones.mp3 (스톤즈)
- engagement-melee.mp3 (전투)
- capnjohnhornpipe.mp3 (배)
- gargoyles.mp3 (가고일)
- dungeon.mp3 (던전)
- unification.mp3 (엔딩)

[설치 방법]
1. 게임 폴더 아래에 "music" 폴더를 만듭니다.
2. MP3 파일들을 music 폴더에 복사합니다.
3. nuvie.cfg에서 다음과 같이 설정합니다:

<ultima6>
  <music>mp3</music>
  <music_dir>ULTIMA6/music</music_dir>
</ultima6>

[참고]
- MP3 파일이 없거나 찾을 수 없는 곡은 자동으로 AdLib으로 재생됩니다.
- OGG, WAV 형식도 지원됩니다.


■ 조작법
--------------------------------------------------------------------------------
[기본 조작]
- 방향키/숫자패드: 이동
- Enter: 행동/대화
- ESC: 메뉴/취소

[명령어]
- A: 공격 (Attack)
- C: 주문 (Cast)
- T: 대화 (Talk)
- L: 살펴보기 (Look)
- G: 집기 (Get)
- D: 버리기 (Drop)
- U: 사용 (Use)
- R: 휴식 (Rest)
- Z: 상태 (Stats)
- Q: 저장 후 종료
