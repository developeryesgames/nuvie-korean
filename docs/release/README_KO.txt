        울티마 6: 거짓 예언자 - 한글화 패치 v1.0
        Ultima VI: The False Prophet - Korean Localization
================================================================================

■ 소개
--------------------------------------------------------------------------------
1990년 오리진 시스템즈에서 발매된 명작 RPG "울티마 6"의 한글화 패치입니다.
Nuvie 엔진을 기반으로 제작되었습니다.


■ 번역 내용
--------------------------------------------------------------------------------
- NPC 대화: 200명 이상, 7,700줄 이상
- 아이템/오브젝트 설명: 600개 이상
- 마법 주문 이름: 121개 전체
- 책, 표지판, 두루마리: 127개
- UI 텍스트, 전투 메시지, 시스템 메시지 전체
- 인트로/엔딩 시퀀스


■ 특징
--------------------------------------------------------------------------------
● 한글 키워드 입력 지원
  NPC와 대화할 때 한글로 키워드를 입력할 수 있습니다.
  예: "이름", "일", "합류", "안녕" 등 (770개 이상 지원)

● 한글 조사 자동 처리
  받침 유무에 따라 을/를, 이/가, 은/는이 자동으로 적용됩니다.

● 전용 한글 폰트
  16x16, 32x32 비트맵 한글 폰트가 포함되어 있습니다.


■ 필요 사항
--------------------------------------------------------------------------------
- 울티마 6 정품 게임 파일
  (GOG.com 등에서 구매 가능: https://www.gog.com/game/ultima_6)
- Windows 10/11


■ 설치 방법
--------------------------------------------------------------------------------
1. 이 패치 파일의 압축을 해제합니다.

2. ULTIMA6 폴더에 정품 게임 파일을 복사합니다.
   필요한 파일:
   - CONVERSE.A
   - CONVERSE.B
   - MAP
   - OBJLIST
   - 기타 게임 데이터 파일들

3. nuvie.exe를 실행합니다.

4. 한글 모드가 기본으로 활성화되어 있습니다.


■ 설정 변경
--------------------------------------------------------------------------------
nuvie.cfg 파일을 편집하여 설정을 변경할 수 있습니다.

한글 모드 끄기:
<korean_mode>false</korean_mode>


■ 조작법
--------------------------------------------------------------------------------
[기본 조작]
- 방향키/숫자패드: 이동
- Enter: 행동/대화
- ESC: 메뉴/취소

[명령어]
- A: 공격 (Attack)
- C: 주문 (Cast)
- T: 대화 (Talk)
- L: 살펴보기 (Look)
- G: 집기 (Get)
- D: 버리기 (Drop)
- U: 사용 (Use)
- R: 휴식 (Rest)
- Z: 상태 (Stats)
- Q: 저장 후 종료
