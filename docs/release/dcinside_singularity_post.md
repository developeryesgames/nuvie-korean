# 디시인사이드 특이점 갤러리 배포글

## 제목
1인 개발자가 AI(클로드)로 고전 RPG 대작 40시간 만에 풀번역한 후기

## 본문

```

특갤러들 안녕
게임 개발자인데 AI 활용 능력 테스트 겸 해본 프로젝트 공유함

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
뭘 했냐면
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1990년작 울티마 6 전체 한글화
- NPC 200명 이상, 대사 7,700줄
- 아이템 600개
- 마법 121개
- 책/표지판 127개
- UI, 전투, 시스템 메시지 전부

이거 옛날 같았으면 팀 단위로 몇 달 걸릴 작업임


━━━━━━━━━━━━━━━━━━━━━━━━━━━━
AI 워크플로우
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

사용 도구: Claude (Opus 4.5 + Sonnet)

1. 대사 추출
   - 원본 게임 데이터에서 Python으로 텍스트 추출
   - NPC별로 분류

2. 컨텍스트 주입
   - "이건 중세 판타지 세계관이고, 말투는 고풍스럽게"
   - "울티마 시리즈의 전통적인 덕목 체계 유지"
   - NPC 성격/직업에 맞게 번역

3. 일괄 번역
   - 한 번에 수백 줄씩 처리
   - 맥락 유지하면서 번역

4. 엔진 수정
   - 한글 폰트 렌더링 구현
   - 한글 키워드 입력 시스템
   - 조사 자동 처리 (을/를, 이/가, 은/는)

5. 테스트/수정
   - 플레이하면서 어색한 부분 수정
   - AI에게 다시 돌려서 개선


━━━━━━━━━━━━━━━━━━━━━━━━━━━━
실제 작업 시간
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

총 40시간 (수면 포함)

- 환경 세팅, 분석: 6시간
- 번역 작업: 14시간
- 엔진 수정: 6시간
- 테스트/디버깅: 2시간
- 잠자기: 12시간

실질 이틀 컷 ㅋㅋ
솔직히 AI 없었으면 시작도 안 했을 듯
7,700줄 번역을 사람이 어떻게 함


━━━━━━━━━━━━━━━━━━━━━━━━━━━━

AI가 잘하는 것:
- 대량 텍스트 일괄 번역
- 일관된 말투/톤 유지
- 맥락 파악하고 적절한 번역
- 코드 수정 (C++, Lua 다 해줌)

AI가 못하는 것:
- 게임 특유의 뉘앙스 (결국 사람이 검수해야 함)
- 레이아웃/UI 문제 (글자 잘림 등)
- 테스트 플레이 (이건 직접 해야 함)

결론

특이점 왔다고 하긴 좀 그렇고
근데 확실히 1인 개발자의 생산성이 미친듯이 올라감

예전에는 "한글화 하고 싶다" → "팀 모아야 하는데..." → 포기
지금은 "한글화 하고 싶다" → AI한테 시킴 → 40시간 만에 완성

이런 식으로 예전에는 팀이 필요했던 작업을
혼자서 처리할 수 있게 된 게 진짜 체감됨
```

## 추가 팁

### 댓글 대응
- "번역 퀄리티 어떰?" → "AI가 1차로 하고 내가 검수함. 어색한 거 있으면 알려주면 AI 다시 돌릴 예정"
- "다른 게임도 할 수 있음?" → "시간만 있으면 가능. 텍스트 추출만 되면 AI가 다 해줌"
- "본업이 뭐임?" → "게임 개발자. AI 활용 테스트 겸 해봄"
